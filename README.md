# Excel-File-Merger
This is a simple Python script that allows you to merge multiple Excel files and their sheets into a single Excel file.

## Getting Started

### Prerequisites

To run this script, you need to have Python and the pandas library installed. You can install the necessary library using the following command:

```bash
pip install pandas
```
## Installation
1. Clone the repository:

```console
git clone https://github.com/haironmysoup/excel-file-merger.git
```
2. Navigate to the project directory:

```console
cd excel-file-merger
```
3. Place your Excel files in the folder where the script is located.

## Usage
1. Make sure your Excel files are located in the same folder as the script.

2. Open a terminal or command prompt.

3. Run the script:
```console
python merge.py
```
4. The merged data will be saved in a file named merged_file.xlsx
